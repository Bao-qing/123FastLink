## 功能介绍


### 📁 秒传链接生成  
将网盘中的文件转换为秒传链接，便于分享。  
- 选择多个文件或文件夹，点击“生成秒传链接”即可快速获取链接。



### 💾 秒传链接保存  
通过秒传链接快速将他人分享的文件转存至你的网盘。  
- 粘贴秒传链接，点击“保存”即可将文件转存到当前目录。  
- 如仅需保存文本记录，可选择“仅保存文本说明”，此时仅会将链接保存为文本文件，不会下载实际内容。

### 🔗 二级秒传链接  
生成更短、更易分享的链接文件。  
> ⚠️ 注意：此功能会先生成原始文件的一级秒传链接，然后自动创建一个包含该链接的文本文件，再基于该文本文件生成二级秒传链接。  
> ⛔ 系统会在当前目录生成一个 `.123fastlink.txt` 文件作为“种子文件”，**⚠️请勿删除**，否则二级链接将失效。“种子文件” ✅️ 可以移动，可以放到一个专门的文件夹，确保不要删除即可

> 📌 建议在使用前先**设置一个种子文件路径**，不设置就在当前路径生成.

进入要设置的路径，复制浏览器地址最后的数字 ID （一般为8位，如：24697788）
//www.123pan.com/?homeFilePath=18648863%2C24697749
注意不要复制到其他参数，如 `%2C` 逗号等。
进入设置页填入该 ID 并保存。
此后生成的二级秒传链接均会使用该路径存放种子文件。

> 💡 使用场景：适合需要分享大量文件时使用，避免链接过长不便传播



### 📦 从秒传文件获取

选择保存包含一级秒传链接的文本文件，脚本会自动读取其中的链接并保存文件。

### 🧠 智能进度显示  
实时展示任务处理进度，并支持随时取消操作。  
- 提供清晰进度条与状态提示  
- 支持中途取消，操作更自由

## 安装指南

### 第一步：安装 Tampermonkey 浏览器插件

Tampermonkey 是一个用户脚本管理器，123FastLink 需要它才能运行。

**Chrome/Edge 浏览器：**
1. 访问 [Chrome Web Store](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
2. 点击 **"添加至Chrome"** 按钮
3. 在弹出框中点击 **"添加扩展程序"** 确认

**Firefox 浏览器：**
1. 访问 [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
2. 点击 **"添加到Firefox"** 按钮
3. 在弹出框中点击 **"添加"** 确认

**Safari 浏览器：**
- 不支持，请使用 Chrome、Edge 或 Firefox

### 第二步：安装 123FastLink 脚本

**方法一：从 GreasyFork 或 ScriptCat 安装（推荐）**
1. 访问 [123FastLink - GreasyFork](https://greasyfork.org/zh-CN/scripts/525210-123fastlink) 或 [123FastLink - ScriptCat](https://scriptcat.org/zh-CN/script-show-page/5041)
2. 点击 **"安装此脚本"** 按钮
3. 在 Tampermonkey 窗口中点击 **"安装"** 确认

**方法二：手动安装**
1. 在 Tampermonkey 中点击 **"创建新脚本"**
2. 复制 `123FastLink.v3.user.js` 或 `123FastLink.v3.debug.js` 的全部内容
3. 粘贴到脚本编辑器中
4. 点击 **"保存"** (Ctrl+S)

### 第三步：验证安装成功

1. 在浏览器地址栏访问 https://www.123pan.com/
2. 登录你的 123网盘 账号
3. 在文件列表上方工具栏中应该能看到 **🚀秒传** 按钮
4. 如果看不到，请刷新页面或检查 Tampermonkey 是否已启用

## 使用步骤

### 场景一：生成秒传链接（分享文件）

#### 步骤 1：登录和选择文件

1. 打开 https://www.123pan.com/ 并登录
2. 进入你要分享的文件所在的文件夹
3. **勾选要分享的文件或文件夹**
   - 点击文件/文件夹前的复选框
   - 或点击 **全选** 复选框选择当前文件夹的所有文件
   - 支持按住 Ctrl 多选或反选

#### 步骤 2：生成秒传链接

1. 点击工具栏中的 **🚀秒传** 按钮
2. 在弹出菜单中点击 **"生成秒传链接"**
3. 脚本开始处理，你会看到进度条：
   - 📋 "正在扫描文件夹..."
   - 🔍 "正在获取文件信息..."
   - ✅ "秒传链接生成完成"

#### 步骤 3：复制和分享

1. 处理完成后，弹出窗口中会显示秒传链接
2. 点击 **"复制"** 按钮将链接复制到剪贴板
3. 粘贴链接分享给其他人：
   - 发送给朋友（QQ、微信、邮件等）
   - 发布到论坛或社交媒体
   - 保存到文本文件备份

### 场景二：保存秒传链接（获取他人分享的文件）

#### 步骤 1：准备秒传链接

1. 从他人那里获取秒传链接
2. 将链接复制到剪贴板

#### 步骤 2：打开保存对话框

1. 登录 https://www.123pan.com/
2. **进入你要保存文件的文件夹**（非常重要！）
3. 点击工具栏中的 **🚀秒传** 按钮
4. 在弹出菜单中点击 **"保存秒传链接"**

#### 步骤 3：粘贴和保存

1. 在文本框中粘贴秒传链接（Ctrl+V）
2. 点击 **"保存"** 按钮
3. 脚本开始处理：
   - 📁 "正在创建文件夹..."
   - 💾 "正在保存第 X / Y 个文件..."
   - ✅ "保存完成"
4. 刷新页面即可看到保存的文件

#### 步骤 4：查看结果

1. 如果所有文件都保存成功，会显示：✅ 成功：X 个文件
2. 如果有文件失败，会显示：❌ 失败：X 个文件
3. 可以点击 **"重试"** 重新保存失败的文件

### 场景三：生成二级秒传链接（更短的分享链接）

#### 步骤 1：选择文件

1. 勾选要分享的文件或文件夹
2. 点击 **🚀秒传** > **"生成二级秒传链接"**

#### 步骤 2：等待生成

1. 脚本会自动：
   - 生成一级秒传链接
   - 将链接保存为网盘内的文本文件
   - 生成该文本文件的二级秒传链接

#### 步骤 3：分享短链接

1. 复制生成的二级秒传链接
2. 分享给他人
3. 其他人保存二级链接时，脚本会：
   - 先下载文本文件（包含一级链接）
   - 自动保存一级链接中的所有文件

### 场景四：导出为 JSON 格式（备份）

1. 生成秒传链接后，点击 **"导出为 JSON"**
2. JSON 文件包含文件的完整信息，便于备份和管理
3. 可以在文本编辑器中查看和编辑

## 功能详解

### 秒传链接是什么？

秒传链接是一种特殊的分享格式，包含了文件的：
- 📄 文件名和路径
- 📊 文件大小
- 🔐 文件的唯一标识（MD5 哈希值）

**核心特点：**
- ⚡ **极速：** 不需要重新上传文件内容
- 🔗 **持久：** 理论上永远有效（只要原文件还在网盘上）
- 📦 **完整：** 保留文件夹结构和所有属性

### 秒传的工作原理

```
你的文件 → 提取信息 → 生成链接
                          ↓
他人收到 → 验证文件 → 快速保存（不上传，直接复用）
```

如果网盘上已经存在相同 MD5 的文件，就直接链接过来，无需重新上传！

### 进度显示详解

生成链接时，你会看到不同的进度阶段：

| 进度 | 说明 |
|------|------|
| 📋 扫描文件夹 | 正在遍历文件夹和子文件夹 |
| 🔍 获取文件信息 | 正在获取每个文件的详细信息 |
| 📝 生成链接 | 正在转换为秒传链接格式 |
| ✅ 完成 | 秒传链接已生成，可以复制 |

### 文件夹处理

- ✅ 支持多级文件夹
- ✅ 自动保留文件夹结构

## 常见问题

### 什么是秒传链接？为什么要使用秒传链接？
秒传链接允许用户通过文件的唯一标识符（如哈希值）来快速上传文件，而无需实际上传文件内容。对于服务端来说，该文件是由自己上传，属于本人文件存储，可以避免生成分享链接带来的一系列限制问题

### ❓ 我看不到 🚀秒传 按钮怎么办？

**解决方案：**
1. 检查 Tampermonkey 是否已启用（浏览器右上角图标）
2. 确认 123FastLink 脚本在 Tampermonkey 中已启用
3. 尝试刷新页面（F5）
4. 清除浏览器缓存后重试
5. 如果还是不行，尝试在浏览器控制台（F12）中查看是否有错误提示

### ❓ 秒传链接保存失败怎么办？

**常见原因和解决方案：**

| 错误信息 | 原因 | 解决方案 |
|--------|------|--------|
| "未能实现秒传" | 网盘中不存在该文件 | 确认原始文件仍在网盘中，或等待一段时间重试 |
| "创建文件夹失败" | 网络错误 | 检查网络连接，稍后重试 |
| "请求失败" | API 请求出错 | 刷新页面，重新登录，或稍后重试 |
| "文件过大" | 超过网盘限制 | 检查文件大小是否超过限制 |

**重试步骤：**
1. 点击保存结果中的 **"重试"** 按钮
2. 或者稍等几分钟后重新保存

### ❓ 如何分享大量文件？

**建议方案：**
- ✅ **一级秒传链接** - 适合 100 个文件以内，复制粘贴即可
- ✅ **二级秒传链接** - 适合任意数量，链接较短，方便分享
- ✅ **分多个链接** - 如果文件过多（>1000个），分成多个链接

### ❓ 秒传链接多久失效？

**答：** 只要：
- ✅ 原始文件仍在网盘中
- ✅ 网盘存在相同 MD5 的文件
- ✅ 链接格式保存正确

即使你删除了原始文件，只要网盘中还有其他人分享的相同文件，秒传链接仍然有效。

### ❓ 能否修改秒传链接的文件名？

**答：** 可以，但需要手动编辑。秒传链接格式：
```
123FLCPV2$path/old_name.txt%etag#size#...
```
修改为：
```
123FLCPV2$path/new_name.txt%etag#size#...
```
修改后仍然有效。

### ❓ 秒传安全吗？会泄露隐私吗？

**答：** 相对安全：
- ✅ 秒传链接只包含文件的公开信息（MD5、大小、路径）
- ✅ 不包含你的账户信息
- ⚠️ 但拥有链接的人可以将文件保存到他们的网盘
- ⚠️ 不要分享包含敏感信息的文件的秒传链接

### ❓ 如何回收已分享的秒传链接？

**答：** 遗憾的是无法直接回收，但可以：
1. **删除原始文件** - 这样其他人就无法通过秒传保存（但如果网盘上有其他人的副本，仍然可能秒传成功）
2. **生成新链接** - 分享新的链接，通知别人不要使用旧链接
3. **备注说明** - 在分享链接时说明时限
